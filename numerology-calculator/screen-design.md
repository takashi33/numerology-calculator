# 画面設計：数秘電卓（仮）

## レイアウト（1画面・縦並び）

```
┌─────────────────────────────┐
│        数秘電卓               │
│  (サブタイトル・簡単な説明)      │
├─────────────────────────────┤
│  生年月日: [____/__/__]       │
│  名前(ローマ字): [__________] │
│                              │
│        [ 計算する ]           │
├─────────────────────────────┤
│  ライフパスナンバー             │
│         ７                   │
│   （一言解説テキスト）           │
│                              │
│  デスティニーナンバー           │
│         ３                   │
│   （一言解説テキスト）           │
└─────────────────────────────┘
```

## 配色・雰囲気
- 数秘術・占い系のアプリらしく、落ち着いた紺〜紫系のグラデーション背景に、金色（ゴールド）のアクセントを使う
- 結果の数字は大きく、丸みのあるカードの中央に表示する
- 断定的にならないよう、解説テキストは控えめなグレーの小さめの文字にする

## 入力フォーム
- 生年月日：ブラウザ標準の日付入力（`<input type="date">`）を使用し、入力ミスを減らす
- 名前：半角英字のみを想定したテキスト入力。プレースホルダーに「例：Taro Yamada」のようなローマ字例を明記する

## 結果カード
- ライフパスナンバー・デスティニーナンバーをそれぞれ別カードで表示
- 数字はカードの中央に大きく表示し、下に一言解説を添える
- マスターナンバー（11・22・33）の場合は、カードに金色の縁取りなど、通常とは違う特別感のある見た目にする

## エラー表示
- 入力不備の場合は、フォーム直下に赤字ではなく落ち着いた色（濃いグレーや紺）でメッセージを表示する（過度に不安を煽らないトーンにする）
